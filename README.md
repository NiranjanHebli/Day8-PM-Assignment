## Part B:- 

Find Pairs :-
Write a program to find all pairs of two numbers in an array that add up to a given sum.

## Requirements

- Python 3.8+ (recommended)

## Files

- `brute_force.py` — main program containing the brute-force algorithm to find pairs of two numbers in an array that add up to a given sum.
- `optimized.py` — main program containing the optimized algorithm to find pairs of two numbers in an array that add up to a given sum.

## Usage

Run the program from the project root directory:

```bash
cd find_pairs
python3 brute_force.py
python3 optimized.py
```
